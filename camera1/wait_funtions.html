wait_funtions module

Various wait functions to be used with time lapse scripts.

camera1.wait_funtions.caputure_continuous_wait_absolute(camera: picamera.PiCamera, wait: int, capture_output: str = '/home/pi/time-lapse/img{timestamp:%Y-%m-%dT%H-%M-%S-%f}.jpg') → None[source]

Capture image and delay so that total time to execute is given wait of seconds. I.e. delay is adjusted by how long it takes for image capture so that absolute time taken is the given wait time, not time taken to capture image + wait time.

Parameters

  • camera (picamera.PiCamera) – The camera object on which capture_continuous is called.

  • wait (int) – Absolute time in seconds that this function is to take to execute

  • capture_output (str) – Value to pass as output parameter to picamera.PiCamera.capture_continuous(). Default: /home/pi/time-lapse/img{timestamp:%Y-%m-%dT%H-%M-%S-%f}.jpg
